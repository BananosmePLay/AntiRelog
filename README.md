Версия плагина: 3.0.12
Под версию сервера: 1.8-1.19.4
Последнее обновление: 19.02.2026

## Особенности:
    Использует BossBar(1.9+), ActionBar и Title.
    Выключает креатив, флай, невидимость, режим бога при атаке игрока.
    Убивает если игрок вышел во время пвп.
    Позволяет игнорировать PVP DENY во время пвп.
    Не позволяет писать команды во время пвп.
    Можно отключить ПВП режим в определённых регионах
    Не позволяет часто есть зачарованные и обычные золотые яблоки.
    Может блокировать телепорты во время пвп.
    Все сообщения настраиваются.
    Есть право для обхода проверок.
    Можно отключить сообщения о заходе, выходе, смерти игрока.

## Права
    antirelog.
    antirelog.bypass - отключает все проверки у игрока
    antirelog.reload - позволяет перезагружать плагин

## Команды
    /anitrelog stop all/Player (время) - Отключает пвп режим у игрока(-ов)
    /antirelog give Player1 Player2 - Выдает пвп режим игрока
    /antirelog reload - Перезагрузить плагин
